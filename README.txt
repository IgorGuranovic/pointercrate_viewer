REQUIREMENTS:
Google Chrome
	Necessary imports:
	pip install helium
	pip install bs4
Python 3.7.6 64-bit ('base': conda) works best in this version
Installing Anaconda would be preferred, since without it, you have to install more packages than the two listed.

HOW TO USE:
Step 1: Download this whole "pointercrate_viewer" repository, and extract it.
Step 2: Open the main.py file to start the program, a GUI will pop up.
Step 3: Write the name of your desired player into the "Player" field. Type it exactly how it is written on pointercrate.com
Step 4: Hit Enter and wait for another GUI to pop up, displaying all of the top 150 levels completed (Green=complete, Red=incomplete).
Step 5: Either repeat steps 3-4 with another player of your choosing or hit Cancel to exit out of the program.
